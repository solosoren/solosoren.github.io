Media Recommendation

Current Project

Media Recommendation is a personal MacOS application that recommends movies, books, and music to users. Users start by rating media that they have read / watched / listened to. The application then uses an external Collaborative Filtering algorithm to recommend media to the user. The Collaborative Filtering algorithm is an external library that I created. It gives the client protocols to follow for their objects. The client can then use these objects with their own data to train a Collaborative Filtering model and get recommendations via the trained model.

SAT Vocab Game

Course Project

C / C++
QtCreator

SAT Vocab Game is a speed typing educational game designed to help people learn SAT vocabulary. When the game starts, enemy words come down the screen to attack the player. The user must type out the words in order to destroy them. The goal is to survive as long as possible before getting hit. The application was written in C / C++. Most of my contributions were on the backend. This included creating most of our GameObject's as well as most of the ObjectController. I worked along an 8 person team running 4 SCRUM week long sprints.

Socialize

Personal Project

iOS Application
Swift
Firebase

Socialize is an iOS Messaging App that allows you to message your iCloud friends using your iCloud account. You can also connect to new people through the app. Socialize was my personal capstone project that I completed for DevMountain, a coding bootcamp in Salt Lake City. I wrote the app in Swift and used Firebase for my database. I also designed the entire app myself. Sketch was used for the initial design. That was then transferred into XCode Storyboards.
